FAQ 2: Should I Use Repl.it or Coding Rooms?

repl.it is a code editor that we are using for the first 15 days of the course. It requires no setup and is easy to use, we will continue to use it for projects and general coding. Coding rooms is an automated coding exercise platform that we'll use to check your code only for the lessons labelled [Interactive Coding Exercise]. Previously, we used repl.it for the coding exercises as well and the links to the repl.it exercises are still in the course resource links. You can choose which you prefer. I personally think that coding rooms has a slightly better UI but it really doesn't matter. What matters is you can write code.


FAQ 3: When I go to Coding Rooms I don't see a Run button.

Try a different browser (I recommend Chrome) and make sure you don't have any VPN enabled or a firewall blocking the website.
