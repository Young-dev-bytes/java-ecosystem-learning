In Step 03 and Step 04 - we create person table and insert data into it.

Spring Boot version 2+ has a minor change.


All DDL(Data Definition Language) - Table Creations should be done in schema.sql

Listing 1 - schema.sql

create table person
(
   id integer not null,
   name varchar(255) not null,
   location varchar(255),
   birth_date timestamp,
   primary key(id)
);


All DML(Data Manipulation Language) - Data Updates should be done in data.sql

Listing 2 - data.sql
